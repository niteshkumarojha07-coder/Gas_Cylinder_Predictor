# Gas Cylinder Predictor App

## Project Description

The Gas Cylinder Predictor App is a mobile application built using Flutter and SQLite. This app aims to help users predict the availability and consumption of gas cylinders in their households. 

### Features:
- User-friendly interface to track gas usage.
- Notifications for gas cylinder availability.
- Integration with SQLite for local storage of user data.
- Predictive algorithms to estimate when the next cylinder will be needed based on usage patterns.

### Technologies Used:
- **Flutter**: For building the UI and ensuring cross-platform compatibility.
- **SQLite**: To store user data locally, allowing for quick access and management of settings.

### Getting Started:
To get a copy of the project up and running on your local machine for development and testing purposes, follow these steps:

1. Clone the repo:
   ```bash
   git clone https://github.com/niteshkumarojha07-coder/gas_cylinder_predictor.git
   ```
2. Navigate to the project directory:
   ```bash
   cd gas_cylinder_predictor
   ```
3. Install the dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
